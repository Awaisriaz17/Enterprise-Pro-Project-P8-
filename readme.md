# Crowd Engagement Prediction System

 Project Overview
This project is a web-based application developed using Flask and Machine Learning. It allows users to upload crowd data (CSV file) and predicts the engagement level (High or Low).



 Features
User login system
CSV file upload
Machine learning prediction
Dashboard interface



Technologies Used
Python
Flask
Pandas
Scikit-learn



Project Structure
